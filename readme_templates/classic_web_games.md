# 🎮 Classic Web Games Collection

A compilation of interactive, browser-based games including a space-nebula themed Tic-Tac-Toe and a retro Rock-Paper-Scissors game. Built with pure HTML5, CSS3, and ES6 JavaScript to demonstrate DOM manipulation, event handlers, and responsive layout styling.

## 🚀 Key Features

- **Tic-Tac-Toe Game:** 
  - Dynamic game state tracking (board arrays).
  - Animated victory path highlighting.
  - Interactive grid cells with custom click logic.
- **Stone-Paper-Scissors Game:**
  - Automated CPU selection choosing random options.
  - Instant score tracking displaying user vs computer wins.
  - Custom win/lose visual notifications.
- **Space-Nebula Visuals:** Styled using CSS gradients, glowing drop shadows, and responsive glassmorphism containers.
- **Full Offline Compatibility:** Runs instantly without requiring database setup or node servers.

## 🛠️ Tech Stack

- **Markup:** HTML5 (semantic layout structures)
- **Styling:** CSS3 (flexbox/grid layouts, keyframe animations, responsive breakpoints)
- **Programming:** JavaScript (ES6, DOM Event Listeners, Game loops)

## 🏁 Getting Started

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/HimanshuSh31/Classic-Web-Games.git
   cd Classic-Web-Games
   ```
2. **Open the game dashboard:**
   - Double-click `index.html` to open it in your browser.
   - Or run a quick server using Python:
     ```bash
     python -m http.server 8000
     ```
     Navigate to `http://localhost:8000` in your web browser.
