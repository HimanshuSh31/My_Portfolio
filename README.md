## 🎨 Advanced Visuals & Interactive Features

- **Interactive Particle Background:** Canvas-based interactive particle network in the hero section that shifts dynamically and responds to desktop mouse movement (magnetic attraction).
- **Sleek Dark Mode Switcher:** A custom-engineered sliding pill toggle with spinning sun/moon icons and `localStorage` caching to remember visitor preferences.
- **Mouse Spotlight Glow:** A custom JavaScript-powered spotlight glow effect (`.glow-card`) that follows the mouse cursor on cards and buttons.
- **Animated Stats Counter:** Eased count-up counter (from 0 to target) triggered automatically as the user scrolls the stats strip into view.
- **Scroll Progress & Sticky Navbar:** A top-screen gradient progress bar indicating read progress, coupled with a sticky glassmorphic navbar.
- **Active Section Highlighting:** Intersection Observer monitoring the viewport to highlight navigation links dynamically as you scroll.
- **Categorized Certification Filter:** Smooth filtering animations letting recruiters group certifications by *Data Analytics*, *Cloud & AI*, or *IoT & Hardware*.
- **Contact Form Validation & Toasts:** Fully validated contact form integrated with Formspree, featuring custom slide-up toast notification alerts.

## 🛠️ Tech Stack

- **HTML5:** Semantic architecture for structured document indexing and optimal SEO.
- **CSS3 / Vanilla CSS:** Custom scrollbar stylings, mouse-glow gradients, hamburger-to-X transitions, staggered entry delays, and dark mode overrides.
- **JavaScript (ES6+):** Scroll animations, Intersection Observers, stats counting logic, typing headlines, canvas particles, and contact submissions.
- **Tailwind CSS:** Utility-first layout styles via CDN (fully optimized standard classes).
- **Fonts & Icons:** Google Fonts (*Outfit* & *Inter*) and Font Awesome 6.5.

## 🏁 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Safari, Edge, Firefox).

### Run Locally
1. **Clone the repository:**
   ```bash
   git clone https://github.com/HimanshuSh31/My_Portfolio.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd My_Portfolio
   ```
3. **Open the site:**
   - Double-click `index.html` to open it in your browser.
   - Or serve it locally using Python:
     ```bash
     python -m http.server 8000
     ```
     Then navigate to `http://localhost:8000`.
